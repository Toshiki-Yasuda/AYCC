概要

介護施設や医療機関の現場で発生するシフト作成業務を大幅に効率化するWebアプリケーションです。 Google Apps Script上で動作し、Google スプレッドシートをデータベースとして使用するため、追加コスト不要で導入できます。

20名規模の施設を想定し、看護師・准看護師・介護士・事務など複数の役職に対応。 スタッフの希望休・人員配置条件・連勤制限・同時勤務NGペアなど、現場のリアルな制約を反映したシフト自動生成が可能です。

主な機能

  • ●
    シフト自動生成（3段階アルゴリズム）

    ①希望反映 → ②人員基準充足 → ③残り自動割当の3段階で、公平性スコアリングにより偏りのないシフトを生成

  • ★
    スタッフ専用URL希望提出 NEW

    各スタッフに専用URLを発行。ログイン不要、スマホから簡単に希望を提出

  • ●
    管理画面からの希望入力

    カレンダー形式で希望休・夜勤希望・有給などを直感的に入力（管理者代理入力）

  • ●
    人員配置条件

    役職ごと・シフトタイプごと・平日/休日ごとに最小・最大人数を設定

  • ●
    同時勤務NGペア

    相性や業務バランスを考慮し、特定のスタッフ同士を同日・同シフトに配置しない制約

  • ●
    セル編集・Excel/PDF出力

    生成後のシフト表を手動で微調整し、Excelファイルや印刷用PDFとして出力

  • ●
    集計・基準チェック

    日勤・夜勤・早番・遅番の日別人数を自動集計し、基準未達を警告表示

導入のメリット

追加コスト不要

Google Workspace環境があればすぐに導入可能。サーバー費用がかかりません。

データはスプレッドシート

馴染みのあるスプレッドシートにデータが蓄積されるため、既存業務との連携も容易です。

スマホ完全対応

レスポンシブデザイン＋週表示切替で、スマホでも見やすく操作しやすい設計。ダークモードにも対応。

カスタマイズ対応

施設の運用に合わせたシフトタイプ追加・条件変更などの個別対応が可能です。

使用技術

Google Apps Script Google Sheets HTML / CSS / JavaScript clasp

サーバー費用ゼロ、Googleアカウントだけで運用開始できます。

施設の規模や運用に合わせたカスタマイズも承ります。
