tmuxの基本操作:
Ctrl+b → キー
まず Ctrl+b（プレフィックスキー）を押してから、続けてコマンドキーを押す
セッション ウィンドウ ペイン 便利な操作
1
セッション管理
作業環境の起動・復帰・切り離し
操作コマンド / キー
新規セッション作成tmux new -s dev
セッション一覧tmux ls
セッションに復帰tmux a -t dev
セッションを抜ける（維持）Ctrl+b → d
セッション切替メニューCtrl+b → s
セッション名を変更Ctrl+b → $
セッションを終了tmux kill-session -t dev
💡
detach（Ctrl+b → d）はセッションを「切り離す」だけ。VPS上でセッションは動き続けるので、別のデバイスから tmux a -t dev で即復帰できます。
2
ウィンドウ操作
タブのように切り替える作業スペース
操作キー
新規ウィンドウCtrl+b → c
次のウィンドウCtrl+b → n
前のウィンドウCtrl+b → p
直前のウィンドウに戻るCtrl+b → l
番号で移動Ctrl+b → 0〜9
ウィンドウ名変更Ctrl+b → ,
ウィンドウ一覧（ツリー表示）Ctrl+b → w
ウィンドウを閉じるCtrl+b → &
💡
Ctrl+b → w でウィンドウ一覧がツリー形式で表示されます。矢印キーで選択してEnterで移動。複数セッション間の移動にも使えます。
3
ペイン（画面分割）
1つのウィンドウを分割して並行作業
操作キー
左右に分割Ctrl+b → %
上下に分割Ctrl+b → "
ペイン間移動Ctrl+b → 矢印キー
次のペインへ移動Ctrl+b → o
ペインサイズ変更Ctrl+b 長押し + 矢印キー
レイアウトを順に切替Ctrl+b → Space
ペイン番号を表示Ctrl+b → q
ペインの全画面切替Ctrl+b → z
ペインを閉じるexit または Ctrl+d
💡
zoom（Ctrl+b → z）で1ペインを全画面表示。もう一度押すと元のレイアウトに戻ります。Claude Codeの出力を広く見たいときに便利。
+
便利な操作
スクロール・コピー・コマンドモード
操作キー
スクロールモード開始Ctrl+b → [
スクロール中の移動↑ ↓ PgUp PgDn
スクロールモード終了q または Esc
コマンドモードCtrl+b → :
時計を表示Ctrl+b → t
キーバインド一覧Ctrl+b → ?
💡
スクロールモード（Ctrl+b → [）はClaude Codeの長い出力を遡りたいときに必須。矢印キーやPgUp/PgDnで自由にスクロールできます。
