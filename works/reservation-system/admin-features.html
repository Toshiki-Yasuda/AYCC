管理者向け機能

Admin Features

教室運営に必要なすべてを、ひとつの画面から。

スマホで完結 — ほぼ全ての操作がスマートフォンから可能です。

ダッシュボード

月間カレンダーで予定を一目で確認。直近の予約一覧と通知センターで、教室の状況をひと目で把握できるホーム画面です。

✓

月間カレンダー表示

✓

直近の予約一覧

✓

通知センターで重要なお知らせを表示

✓

統計カードタップで通知連動アニメーション

予約管理

予約の一覧表示、承認、キャンセル処理がワンタップで完了。振替処理もスムーズなワークフローで対応できます。予約状況の確認から対応まで、すべてスマホで。

✓

予約一覧・承認・キャンセル

✓

振替処理のワークフロー

✓

ワンタップで操作完了

スロット管理

レッスン枠の作成・削除を直感的に操作。日時、講師、レッスン内容を設定するだけで、新しいスロットが生成されます。スケジュール変更も柔軟に対応できます。

✓

レッスン枠の作成・削除

✓

日時・講師・内容の設定

✓

柔軟なスケジュール変更

固定スケジュール

曜日固定のレッスンを設定すれば、毎週自動的にスロットが反映されます。定期レッスンの管理がシンプルになり、スケジュール作成の手間を大幅に削減できます。

✓

曜日固定レッスンの設定

✓

定期レッスンの自動反映

✓

スケジュール作成の手間を削減

休講日管理

祝日や教室の都合による休講日を設定。該当日のスロットは自動的に非表示となり、生徒への影響を最小限に抑えます。休講日の一括管理で運営がスムーズに。

✓

休講日の設定

✓

該当日のスロット自動非表示

✓

生徒への影響を最小限に

生徒管理

生徒情報の登録・編集、メール送信が一元管理できます。有効な生徒と無効な生徒がセクション分割され、ふりがな対応で検索もスムーズ。男女で色分けされたカード表示で視認性が向上。誕生日通知機能により、生徒一人ひとりへのきめ細やかな対応が可能です。

✓

生徒情報の登録・編集

✓

メール送信機能

✓

男女カード色分け表示

✓

誕生日通知機能

✓

有効/無効セクション分割

✓

ふりがな対応（生徒名・保護者名）

設定

教室の基本情報、利用規約の編集、プレビュー機能、ガイドを備えた4つのタブで構成。変更内容はリアルタイムでプレビューできるため、安心して設定を更新できます。

操作ログ

システム内のすべての操作履歴を記録。エラーレベル表示により、問題の早期発見と対応が可能です。トラブルシューティングや運用改善に役立つ機能です。

✓

操作履歴の確認

✓

エラーレベル表示

✓

問題の早期発見

生徒検索

生徒の名前・ふりがな・保護者名で素早く検索。大人数の教室でも、目的の生徒をすぐに見つけることができます。

✓

名前・ふりがな・保護者名で検索

✓

リアルタイム絞り込み

✓
