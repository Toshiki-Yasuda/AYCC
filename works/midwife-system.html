概要

出張専門の助産院向けに、ホームページと業務管理システムをセットで提供するソリューションです。 「予約受付」「顧客・お子様管理」「電子カルテ」「スケジュール管理」を一つのシステムに集約し、 助産師さんの訪問業務を効率化します。

ホームページは Next.js + Tailwind CSS で構築し、Cloudflare Pages でホスティング。 管理システムは Google Apps Script 上で動作するため、サーバー維持費用がかかりません。 訪問先のスマートフォンからもカルテ入力・予約確認が可能です。

提供内容

ホームページ

  • ● トップページ（院の紹介・サービス概要・補助金案内）
  • ● 助産院紹介・助産師プロフィール
  • ● メニュー・料金（訪問の流れ・FAQ付き）
  • ● アクセス・お問い合わせ
  • ● レスポンシブ対応（PC / スマホ）

業務管理システム

  • ● 予約管理（受付・確認・リマインダー通知）
  • ● 顧客・お子様情報の紐づけ管理
  • ● 電子カルテ（メニュー別テンプレート）
  • ● スケジュール管理（カレンダー表示）
  • ● スマートフォン完全対応

管理システムの主な機能

  • ●
    ダッシュボード

    今日の予約・確認待ち・今月の予約数を一覧表示。日々の業務状況をひと目で把握できます。

  • ●
    予約管理

    Webフォームからの予約受付・承認・キャンセル管理。ステータス変更時に自動メール通知。前日のリマインダー配信で予約忘れを防止。

  • ●
    顧客・お子様管理

    お母さんの情報にお子様を紐づけて管理。出産日・月齢を自動計算し、メニュー提案や記録に活用。複数のお子様にも対応。

  • ●
    電子カルテ

    母乳相談・睡眠相談・ベビーマッサージ・離乳食相談など、メニューに応じたテンプレートを自動表示。訪問先のスマホからも記録可能。過去の施術履歴を時系列で一覧確認。

  • ●
    スケジュール管理

    カレンダー表示で予約状況を確認。日付クリックでその日の予約詳細を表示。訪問スケジュールの管理に最適。

  • ●
    院の設定管理

    院名・メニュー・料金・営業時間などをシステム上で一元管理。設定変更が即座に反映されます。

使用技術

ホームページ

Next.js 14 Tailwind CSS TypeScript Cloudflare Pages

管理システム

Google Apps Script Google Spreadsheet HTML / CSS / JavaScript GAS MailApp

ホームページは静的サイトとして高速配信。管理システムはGoogle Apps Scriptの標準機能のみで構成し、 外部サーバー不要で運用コストを最小限に。データはGoogle Spreadsheetに保存され、 Googleのインフラ上で安全に管理されます。
